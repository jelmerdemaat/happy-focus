Testing the Happy Focus JS

The focus state and its styles will show and retain on click in current Chrome versions (up to version 58) or IE >= 9 once you style buttons or inputs away from their defaults. In IE9 specifically, default :focus styles even when not applying custom styles. Information last updated may 2017.

Try below. The red border (or any other style you give it) is visible in these browsers on click as well as when using the keyboard to navigate.
